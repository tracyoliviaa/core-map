# CoreMap 🗺️

Dein täglicher Habit-Tracker across 6 Lebensbereiche.

## Features
- ✅ Tasks togglen, hinzufügen, löschen
- 📊 Progress Bars pro Bereich
- 🔥 Streak & Punkte System
- 💾 Automatische Speicherung (localStorage)
- 🌙 Dark Mode
- 🔄 Täglicher Reset um Mitternacht

## Tech Stack
React · Vite · Tailwind CSS · Context API · localStorage

## Setup
npm install
npm run dev
